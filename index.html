python-binary-memcached by jaysonsantos

A pure python module to access memcached via it's binary protocol with SASL auth support.

If you tried to use memcached with authentication and get frusted compiling some tools or if your default library does not support it, this is the perfect choice to run it beautifuly.

Dependencies

None

Install

pip install python-binary-memcached
